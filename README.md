# Flower Shop Management System

A comprehensive Java-based desktop application designed to streamline the operations of a florist business. This project was developed collaboratively as part of a university team assignment, focusing on robust software architecture and user-friendly interface design.

## Key Features
*   **Inventory Management:** Efficiently track flower stocks, bouquets, and seasonal supplies.
*   **Order Processing:** Create, manage, and log customer orders in real-time.
*   **User Roles:** Secure login system with different access levels (Admin vs. Employee).
*   **Data Persistence:** Reliable storage of business information using Java-integrated database management.
*   **Financial Reporting:** Simple tools to view sales history and business performance.

## Team Collaboration
This project was developed by a team of 3 students.

## Learning Outcomes
*   Applied **Object-Oriented Programming (OOP)** principles: Inheritance, Encapsulation, and Polymorphism.
